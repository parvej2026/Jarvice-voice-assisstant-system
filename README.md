# Jarvice-voice-assisstant-system
 JARVIS is a python - based voice assistant that can interact with the user through speech recognization,perform tasks like opening applications,searching on google or wikipedia, playing music randomly,telling jokes,and having small talk.

this project uses speech recognization and text-to-speech(TTS) to provide a hand - free assistant experience similar to Iron man's JARVIS.


## ✪ Features

- Great the user according to the time of the day(morning, afternoon, evening)
- Recognize voice commands using google speech   recognization
- Speak responses using pytts
- Time & date annoucement
- Wikipedia search with spoken summery
- Open websites like google, facebook, youtube
- Play random music from a specified folder
- Open system application: Calculator, Notepad,CMD
- Open calender(Google calender via Browser)
- Tell jokes and respons to basic small talk
- Exit gracefully with a voice command

## ⌨ Requirements

- python 3.11 or higher

## How to run?

1.create a virtual environment(optional but recommended):

```bash
conda create -n NEW python=3.11 -y
```
2.activate virtual environment:

```bash
conda activate NEW
```
3.install required packages:

```bash
pip install -r requirements.txt
```

4.Run the NEW scripts

```
python jarvis.py
```
